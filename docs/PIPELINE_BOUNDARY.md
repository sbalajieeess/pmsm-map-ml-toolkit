# Solver/toolkit boundary

Solver repositories own geometry, materials, winding assignment, mesh generation, solver
execution, convergence, skew aggregation, and raw-field extraction.

This toolkit owns canonical validation, common plotting, MTPA/MTPV selection, controller
dataset generation, NN training, pruning, quantization, C export, and cross-solver comparison.

A change to the canonical schema or sign convention requires a tagged toolkit release and
corresponding version update in both solver repositories.
